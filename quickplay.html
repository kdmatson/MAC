QUIZ & INTERLUDES

Questions about the science of the game appear during play. First attempt correct scores maximum points. Choose mode before starting:

ON CHAOS MODE NERD MODE OFF

On = one question per level · Chaos = mid-game surprises too · Nerd = quiz only, no maze · Off = just play

Short animated science scenes — interludes — appear between levels (and once mid-session in Nerd mode). Press NEXT to continue. LEARN MORE opens the source paper.

GOOD TO KNOW

Microbes regrow during gameplay. Microbes flicker before dividing (binary fission). Resistance can also spread to neighboring cells via conjugation.

Resistant microbes glow red (two hits to clear) or gold (one hit left). After eating an Rx pill, a violet pulse shows which survived.

Side tunnels wrap around — viruses slow down inside them.

Watch the three bars above the maze:

Host Quality (Parasite Growth)
Host species + current microbe type — higher = faster regrowth
Viral Virulence (Attack)
Rises each level — viruses become faster and more aggressive
AMR (Antibiotic Resistance)
Rx pill use + background creep — sets how many resistant microbes appear each level

MAD MACs is a vintage-style educational arcade game featuring quiz elements and animated interludes that link to background sources.
